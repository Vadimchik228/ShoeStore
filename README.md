# Интернет-магазин обуви

![ER-диаграмма онлайн-магазина](https://github.com/Vadimchik228/ShoeStore/blob/main/ER-diagramm.png)

## Требования к проекту
* На главной странице размещена основная информация об интернет-магазине: 
    * краткое описание,
    * небольшая товарная галерея, 
    * отзывы клиентов, 
    * контактные данные.
* На верхней панели есть вкладки: 
    * мужчинам, 
    * женщинам, 
    * детям.  
* При выборе вкладки открывается страница с категориями товара (кроссовки, туфли и тд).
* При выборе категории товара открывается список товаров соответствующей категории.
* Под каждым товаром указаны бренд и цена. 
* Пользователь может отсортировать список по возрастанию и убыванию цены. 
* При выборе товара открывается страница с подробным описанием (материал и прочее). Пользователь может:
    * указать размер, 
    * добавить товар в корзину,
    * прочитать отзывы других клиентов.
* При переходе в корзину пользователь может осуществить покупку выбранных товаров (необходимо авторизироваться). 
* Пользователь должен иметь возможность залогиниться на сайте. 
* Необходимо предусмотреть возможность входа на сайт с правами администратора. 
* Администратор имеет следующие возможности: 
    * добавление товара или категории товаров, 
    * изменение товара, 
    * удаление товара или категории товаров, 
    * редактирование основной информации об интернет-магазине.
